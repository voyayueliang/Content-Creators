# 去 AI 味检查清单

## 检查目标

不是把文字修得更漂亮，而是让内容重新像一个具体的人在具体语境里说话。

## 常见 AI 味

- [ ] 过度工整的“不是 A，而是 B”
- [ ] 每段都用同一种起承转合
- [ ] 没有证据就升到宏大判断
- [ ] 用“值得关注”“具有重要意义”等空泛词
- [ ] 反复出现“具体问题”“真实场景”“长期价值”等套话
- [ ] 把读者早就知道的事讲得很郑重
- [ ] 缺少人物、场景、关系和语气
- [ ] 结尾突然拔高
- [ ] 每个段落都像总结，不像人在表达

## 修改方式

- 把抽象判断换成具体场景。
- 把万能句换成作者自己的判断。
- 删除没有信息增量的过渡句。
- 保留必要的口语感和不完美。
- 不强行总结成金句。
- 不用同一个结构连续展开多个段落。
