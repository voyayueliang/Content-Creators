# 活动复盘转公众号 Markdown Skill

## 状态

可对外展示 / GitHub 已发布

## 一句话

把活动复盘、公开报道和现场材料整理成公众号 Markdown 与会后传播包。

## 目标用户

- 活动组织者
- 社区主理人
- 品牌活动负责人
- 伙伴传播负责人

## 真实痛点

活动结束后，材料散落在录音、群聊、照片、伙伴记录和公开报道里。如果不及时整理，现场价值很快沉没。

## 交付承诺

将活动口述复盘、同场公开报道、伙伴记录和图片整合成公众号 Markdown、会后传播包、社区记忆包或伙伴回顾包。

## 最小交付物

- 公开资料调研地图
- 战略叙事诊断
- 公众号 Markdown 草稿
- 图片位置与图注建议
- 发布前确认
- 活动资产回收卡

- 已形成完整 README 和 Skill 文件结构
