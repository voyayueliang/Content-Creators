Moon AI Toolkit

不是文档列表，是可以直接选用的工作流。

每个工具都从真实项目里沉淀出来。选一个入口，准备对应材料，复制启动句，就可以开始跑。

月亮是一个深度内容创作者，长期做访谈、活动报道、播客/直播整理和社区现场写作。这个工作台从她自己的交付过程里长出来：很多项目有录音、策划案、会议纪要和编辑反馈，却缺少一套能判断内容形态、保留人的声音、控制事实风险并复用素材的方法。

快速选择

搭个人内容工作台 做活动报道 / 预告 / 复盘 写公开资料人物侧写 整理播客、音频或视频回放 发布前校对和去 AI 味

按场景选

先看你手上有什么，再选对应工作流。每个入口都保留了启动句，方便直接复制给 AI 开始协作。

Profile Writing

公开资料人物深度侧写

把公开访谈、播客、演讲、文章和社媒材料，转成有事实约束和人类判断的人物稿。

查看产品卡

适合

  • 公开资料研究型人物稿
  • Founder / builder profile
  • 作品集样稿
  • 人物故事工作坊

准备

  • 人物姓名
  • 播客、访谈、演讲或文章链接
  • 目标读者和发布渠道
  • 不能推断或不能公开的边界

产出

  • 人物资料查看台
  • 阅读透镜和文章框架
  • 逐节草稿
  • 引用核查与双门检查

启动句

打开公开资料人物深度侧写工作流。

对象：[人物姓名]
模式：[第三方无采访 / 当事人自述 / 三方协作]
已有素材：[链接 / 文件 / 暂无，请帮我搜集]
发布渠道：[公众号 / Substack / 作品集 / 机构投稿 / 不确定]
大致字数：[比如 3000-6000 字 / 先做框架]

要求：
先不要写完整长文。
请先做人物资料查看台和素材解剖，然后用短问题问我阅读透镜、读者画像和框架判断。
最终交付前请过去 AI 味门和信息校对 / 防幻觉门。
