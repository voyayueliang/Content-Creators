# 社区活动内容传播工作台

## 状态

可对外试点 / Datawhale 首个验证案例 / GitHub 已发布

## 一句话

把社区活动材料变成可审核、可交付、可复盘的内容资产。

## 目标用户

- AI 社区
- 开源社区
- DevRel 团队
- 教育社群
- 技术大会 / 分论坛
- 社区运营
- 活动负责人
- 实习生 / 初级内容运营

## 试点案例

Datawhale 内容传播工作台是这个产品的第一个试点版本。Datawhale 不是产品边界，而是验证场景。

## 真实痛点

社区和活动型组织有很多小型传播需求，材料来自策划案、会议纪要、聊天记录、嘉宾议程、现场反馈和合作方口径。核心编辑没有精力处理所有小稿，但商业合作内容又不能出错。

## 交付承诺

从材料到内容资产，不只生成初稿，还输出发布前确认、商业合作审核卡、当前稿校对、编辑交接和发布后复盘。

## 最小交付物

- 需求收口卡
- 文章方向卡
- 公众号草稿
- 当前稿校对包
- 商业合作审核卡
- 发布前确认清单
- 发布后复盘卡
