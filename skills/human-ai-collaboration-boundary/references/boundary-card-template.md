# Boundary Card Template

Use this template before drafting.

```text
## 这次任务是什么
- 内容类型：
- 使用场景：
- 读者：
- 发布主体：

## AI 可以帮你省哪些力
- 材料整理：
- 事实抽取：
- 结构候选：
- 表达候选：
- 风险标注：

## 必须由人判断的部分
- 角度：
- 公开边界：
- 关系分寸：
- 战略口径：
- 最终声音：
- 是否发布：

## 不能直接公开的信息
- 隐私：
- 内部材料：
- 未确认合作：
- 未授权图片 / 引用：
- 敏感关系：

## 需要校对的事实
- 时间：
- 地点：
- 人名 / 头衔：
- 机构关系：
- 数字：
- 链接 / 报名方式：

## 容易出现 AI 味的地方
- 空泛总结：
- 过度顺滑：
- 万能转折：
- 议程排列：
- 后台编辑视角：

## 下一步建议
- 继续写 / 补材料 / 做调研 / 确认边界 / 暂不公开
```

Keep the card compact. It should help the user decide what to do next.
